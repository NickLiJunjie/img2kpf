# KFX Output plugin slot

This repository does **not** ship `KFX Output.zip`.

To enable `--emit-kfx` or `kfx_direct.py` with the default plugin ID:

1. Download the plugin archive yourself.
2. Save the file here as:

   - `img2kpf/plugins/kfx_output/KFX Output.zip`

You can also keep the zip anywhere else and pass its path explicitly:

- `python kpf_generator.py --emit-kfx --kfx-plugin /path/to/KFX Output.zip ...`
- `python kfx_direct.py --plugin /path/to/KFX Output.zip ...`

Download source:

- MobileRead thread: [https://www.mobileread.com/forums/showthread.php?t=272407](https://www.mobileread.com/forums/showthread.php?t=272407)
